## ✨ Features

- **Ambient Background Motion** — Floating gradient blobs with interactive mouse-following effect
- **Kinetic Typography** — Staggered hero text reveal animations
- **Cursor Glow** — Radial light that follows mouse movement
- **Scroll Reveal** — IntersectionObserver-based entrance animations
- **Counter Animation** — Smooth easeOutQuart number counting
- **Frosted Glass Navbar** — Backdrop-blur effect on scroll
- **Infinite Marquee** — Horizontal scrolling skills ticker
- **Responsive Design** — Mobile-first with breakpoints at 480px, 768px, 1024px
- **Clean SVG Icons** — Brand icons from [lobe-icons](https://github.com/lobehub/lobe-icons) CDN + custom inline SVGs
- **Noise Texture Overlay** — Subtle grain for tactile depth
- **Accessibility** — `prefers-reduced-motion` support, semantic HTML

## 🚀 Quick Start

1. Clone the repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/portfolio.git
   ```
2. Open `index.html` in your browser — no build step required.

## 📂 Structure

```
portfolio/
├── index.html      # Single-file portfolio (HTML + CSS + JS)
├── README.md       # This file
├── LICENSE         # MIT License
└── assets/         # Screenshots & project images (add your own)
```

## 🌐 Deploy to GitHub Pages

1. Go to your repo **Settings** → **Pages**
2. Under **Source**, select `main` branch and `/ (root)`
3. Click **Save** — your site will be live at `https://YOUR_USERNAME.github.io/portfolio/`

## 🛠 Customization

- **Colors** — Edit CSS custom properties in `:root`
- **Content** — Update text directly in the HTML
- **Projects** — Replace thumbnail gradients with real images using `<img>` tags
- **Links** — Update `href` attributes for email, LinkedIn, Behance
